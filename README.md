# Employee-Leave-Management-System
A simple and responsive Employee Leave Management System built using HTML, CSS, JavaScript, and JSON. This project streamlines the leave request process by allowing employees to apply for leave, track their requests, and enabling managers to review and approve or reject applications through an intuitive dashboard.
## Features

* Secure employee login interface
* Apply for different types of leave
* View leave request history and current status
* Manager dashboard for approving or rejecting leave requests
* Dashboard displaying leave statistics and summaries
* Responsive and user-friendly interface
* JSON-based data storage for demonstration purposes
* CRUD operations for managing leave requests

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* JSON

## Project Highlights

* Clean and modular project structure
* Responsive UI suitable for desktop and mobile devices
* Client-side form validation
* Dynamic data rendering using JavaScript
* JSON data handling and manipulation
* Well-organized code following best practices

## Learning Outcomes

This project demonstrates practical frontend development skills, including DOM manipulation, JSON data management, CRUD operations, responsive UI design, and building a business-oriented application that simulates a real-world leave management workflow.
